Upload Limits & Opt-Out Test

This page tests the new upload limit and opt-out functionality.

Test Features:

  • 3 Upload Attempts Limit: Each team can only upload 3 times
  • Opt-Out Functionality: Teams can opt out of uploads permanently
  • UI Updates: Shows remaining attempts and opt-out status
  • Directory Selection: Select entire directories instead of drag-drop

Test Results:

Manual Testing Required:
1. Start the server: node server.js
2. Register a new team or login with existing team
3. Try uploading images multiple times (should max at 3 attempts)
4. Test opt-out functionality
5. Verify UI shows correct remaining attempts

Expected Behavior:

Upload Attempts:
• First upload: Attempts counter shows "1 / 3"
• Second upload: Attempts counter shows "2 / 3"
• Third upload: Attempts counter shows "3 / 3"
• Fourth attempt: Should be blocked with error message
Opt-Out:
• Click "OPT OUT" button
• Confirmation dialog appears
• Upload zone disappears
• "Opted Out" banner appears
• Cannot upload after opting out

Database Schema Changes:

New Team Fields:
• upload_attempts_used: Number of attempts used
• max_upload_attempts: Maximum attempts allowed (default: 3)
• opted_out: Boolean flag for opt-out status
• opt_out_timestamp: When user opted out
• remaining_upload_attempts: Virtual property (calculated)
